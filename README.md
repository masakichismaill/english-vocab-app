# 英単語帳アプリ

HTML / CSS / JavaScript で作成した、英単語の登録・管理・暗記学習ができるWebアプリです。
教師として英語学習の反復練習の重要性を感じた経験をもとに制作しました。

## 主な機能

- 単語の追加・編集・削除（品詞・意味・例文・画像の登録）
- お気に入り登録・苦手フラグ管理
- クイズ機能（登録単語から出題）
- フラッシュカード機能
- 例文管理ページ
- JSONエクスポート / インポート（データのバックアップ・移行）

## 使用技術

- HTML
- CSS
- JavaScript

## 工夫した点

- 単語・品詞・意味・例文・画像を一括管理できる設計にした
- クイズとフラッシュカードの2種類の学習モードを用意し、用途に応じて使い分けられるようにした
- JSONの入出力機能でデータの持ち運びができるようにした
